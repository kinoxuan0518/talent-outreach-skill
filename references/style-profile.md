# Style Profile

This file defines the writing style for all outreach messages generated by talent-outreach.

## How to Train

Provide 3-5 examples of outreach messages you wrote and were happy with. The skill will extract
patterns and write them into this file. All subsequent messages follow this style.

## Status

**UNTRAINED** — using sensible defaults. Run "训练风格" with 3-5 samples to personalize.

⚠️ **This file stays local.** It contains your personal writing fingerprint and contact info.
Do not commit the trained version to git.

## Global Tone (Defaults)

```yaml
address:
  chinese_technical: "老师" + "您"
  chinese_unknown: "您好"
  english_technical: first_name directly
  english_unknown: "Hi {first_name}"

hook:
  pattern: "{认可对方的具体工作} + {一句话证明你读懂了} + {为什么和我们相关}"

length:
  email: 3-5 sentences
  short_channel: 1-3 sentences

forbidden:
  - recruiter_jargon: "诚邀加入"、"看机会"、"您的能力很匹配"
  - corporate_pitch: "行业领先"、"Y轮融资"
  - fake_urgency: "机会难得"、"急招"
  - vague_compliments: "您的背景很匹配"、"非常优秀"

call_to_action:
  preferred: "20分钟聊聊"、"方便聊一下吗"
  avoid: "面试"、"发简历"
```
